# wxml-transpiler

**Under development now...**

## Intro

Use  compiler built in Vue.js to transpile wxml.

## Give it a Try

```sh
# install deps
yarn

# start dev dep
yarn run dev:compiler

# run
node newTest/test

# dev
nodemon newTest/test -w packages/vue-template-compiler -w newTest/test.js
```
