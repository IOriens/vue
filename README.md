# wxml-transpiler

**Under development now...**

## Intro

Use  compiler built in Vue.js to transpile wxml.

## Give it a Try

```sh
# Start
## install deps
yarn
## build dep
yarn run dev:compiler
## run
node newTest/test

# Dev 
## autorestart type check
watch -t flow
```
